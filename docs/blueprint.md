# **App Name**: Baby Blue Bubbles

## Core Features:

- Custom Message: Display a personalized message on the main page.
- Animated Bubbles: Create floating, spontaneously moving bubbles of different sizes.

## Style Guidelines:

- Primary color: Light baby blue (#AEE2FF). Evokes a sense of calm and serenity, reminiscent of baby themes.
- Background color: Very pale blue (#E5F5FF), provides a gentle backdrop without overshadowing the content.
- Accent color: Soft lavender (#D0BFFF) to complement the blue and add a touch of sophistication.
- Body and headline font: 'PT Sans' (sans-serif) to give a modern look
- Gentle, smooth animations for the bubbles moving upwards, creating a calming and mesmerizing effect.
- Center the message and make it easily readable.
